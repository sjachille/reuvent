# reuvent

Test repository on Github with a simple Python calculator application.

## Features

- Basic arithmetic operations (add, subtract, multiply, divide)
- Comprehensive unit tests
- GitHub Actions CI/CD pipeline

## Usage

Run the calculator module:

```bash
python calculator.py
```

Run the tests:

```bash
python -m unittest test_calculator.py -v
```

## CI/CD

This repository includes a GitHub Actions workflow that automatically runs tests on every push to ensure code quality.
